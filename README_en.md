Lesson 2: Conditions & Loops 🐍

This lesson covers two fundamental Python concepts: Conditions and Loops. It's designed for beginners who are just starting to learn Python.

Topics Covered
Conditions

 Checking if a number is even or odd
Loops

 while loop
 break statement
 continue statement
 Nested loops — multiplication table example

Note
